#FFmpeg视频播放，解码与编码

## 🚀 运行环境
- **Qt版本**: 5.9.8 (MinGW 64-bit)
- **FFmpeg版本**: [Zeranoe build] 20200306 (git-cfd9a65)  
  *(注：此为Windows 32位动态库版本)*
## 🛠️ 编译步骤
1. 安装 [Qt Creator](https://www.qt.io/download)
2. 打开 `OMO241201.pro`
3. 点击左下角 **▶ 运行**

## ❓ 常见问题
- 报错"FFmpeg not found"？
  - 下载 [FFmpeg静态库](https://www.gyan.dev/ffmpeg/builds/)
